# 구현 기능 목록
- [x] 다리 길이 입력
  - [x] 3이상 20이하
- [x] 다리 생성
  - [x] 각 다리 칸 마다 위아래 두칸으로 이루어짐
  - [x] 생성 시 위아래 둘중 건널 수 있는 칸 선정
    - [x] 랜덤 값을 통해 선정
    - [x] 0 : 아래 칸 -> D로 나타냄
    - [x] 1 : 위칸 -> U로 나타냄
  - [x] 위아래 둘 중 하나만 건널 수 있음
- [ ] 라운드 마다 이동할 칸 입력
  - [x] 라운드는 다리 길이 만큼 진행
  - [x] U, D 둘 중 하나만 입력 가능
  - [x] 건널수 있다면 O, 없으면 X
  - [x] 다리 칸 별로 | 구분 출력
  - [ ] 전부 건너면 게임 끝
- [ ] 게임 종료 및 재시작 입력
  - [ ] 재시작
    - [ ] 건너다 실패 시 기존다리로 계속
  - [ ] 종료 
    - [ ] 총 시도횟수 와 성공여부 출력
