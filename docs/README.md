# 구현 기능 목록
## 1. 입력
* 다리길이 입력 기능
  * 3 ~ 20 사이의 숫자 검증 기능
  * 숫자인지 검증 기능
* 플레이어 이동 칸 입력 기능
  * 문자 검증 기능
* 게임 재시작 입력 기능
  * 문자 검증 기능

## 2. 출력
* 게임시작 문구 출력 기능
* 게임 과정 출력 기능
* 게임 결과 출력 기능
  * 게임 성공 여부 출력 기능
  * 게임 시도 횟수 출력 기능

## 3. 게임
* 0, 1 중 랜덤한 숫자 생성 기능
  * 0 이면 아래칸, 1이면 윗칸
* 다리 이동 기능
  * 다리 이동 검증 기능
* 게임 재시작 기능
* 게임 종료 기능
* 게임 성공 여부 판별 기능
* 시도 횟수 계산 기능
