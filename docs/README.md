### 기능 목록

- 출력
    - 인트로
        - "다리 건너기 게임을 시작합니다."를 출력
        - "다리의 길이를 입력해주세요."를 출력
    - 다리 맵
        - "이동할 칸을 선택해주세요. (위: U, 아래: D)"를 출력
        - 갔던 경로를 O 또는 X를 통해 출력한다

- 입력
    - 숫자를 입력받는다(다리)
        - 숫자가 아니면 예외
        - 3부터 20까지가 들어오지 않으면 예외
    - 이동할 칸을 입력받는다
        - U,D가 아니면 예외

- 게임 플로우
    - 길이에 맞는 다리를 만듬
    - O면 경우라면 다시 반복
    - X면 게임 종료
    - "게임을 다시 시도할지 여부를 입력해주세요. (재시도: R, 종료: Q)"를 출력
    - "최종 게임 결과" 를 출력
    - "게임 성공 여부: 실패"
    - "총 시도한 횟수: 1"를 출력
