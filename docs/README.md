## 필요 기능
- 입력
  - 다리 길이
  - 이동할 칸
  - 재시작 여부

- 출력
  - 게임시작 문구
  - 다리 갈이 입력 안내문구
  - 이동할 칸 입력 안내문구
  - 재시작 안내문구
  - 이동시 다리 건너기 결과
  - 게임 최종 결과
  
- 게임을 시작한다.
- 다리를 생성한다.
  - 한 칸마다 위, 아래 랜덤으로 정한다.
- 다리를 이동한다.
  - 다리와 일치하는 이동인지 체크한다.
- 게임오버가 되었다.
  - 재시작 여부를 묻는다.
  - 게임을 재시작한다.
- 게임을 클리어하였다.
- 게임을 종료한다.
    
## 에러상황
- 다리 길이 입력
  - 숫자가 아님
  - 3부터 20이 아님
- 이동할 칸 입력
  - "U"나 "D"가 아님
- 재시작 여부 입력
  - "R"나 "Q"가 아님
