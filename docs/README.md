## 기능 목록

1. 게임시작 멘트 출력
2. 다리 길이 입력 멘트 출력
3. 다리 길이를 입력받는다
   1. 3~20 사이 숫자 입력
   2. 문자 or 범위 벗어날시 예외처리
4. 다리 길이만큼 이동할 칸을 입력받는다.
   1. U, D 만 입력 가능
   2. 그외의 입력이 들어올시 예외처리
5. 입력 후 이동한 경로 출력
6. 맞춘경우
   1. 4,5의 과정 반복
7. 틀린경우
   1. 게임 재시도 여부 멘트 출력
   2. R, Q중 하나 입력
   3. 그 외 문자 입력시 예외처리
   4. Q 입력한 경우 게임 종료
   5. R 입력한경우 재시도 카운트 1올린 후 동일한 사다리로 4번 부터 다시 시작
8. 게임 성공 여부 출력
9. 게임 클리어시 최종 게임 결과 출력
10. 총 시도 횟수 출력
