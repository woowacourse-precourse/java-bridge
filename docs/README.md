## ✅ 기능구현 목록

- [x] 다리 길이 숫자 입력
    - [x] (예외) 숫자가 아니면 예외
    - [x] (예외) 3~20사이가 아니면 예외
- [x] 랜덤으로 다리 생성
- [x] 이동 명령 U,D 입력
    - [x] (예외) 입력이 U,D가 아니면 예외
- [x] 생성된 다리와 비교해 정답 판정
    - [x] 정답이면 다리 끝 까지 동일하게 진행
    - [x] 틀리면 게임 다시 시작 할지 Q,R 입력
        - [x] (예외) Q,R이 아니면 예외
-[x] 결과 출력
