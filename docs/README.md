## 구현 리스트 ##

- 입력
  - 다리 길이 입력
    - 3~20이 아닐 시 예외 처리
  - 라운드 마다 플레이어가 이동할 칸 입력
    - U / D만 가능 아닐 시 예외 처리
  - 게임 재시작 / 종료 입력
    - R / Q만 가능 아닐 시 예외 처리
- 출력
  - 게임 시작 문구
  - 게임 종료 문구
  - 다리 출력
  - 최종 게임 결과
    - 최종 다리 출력
    - 총 시도한 횟수 출력
    - 게임 성공 여부
    - 총 시도한 횟수
- 기능
  - BridgeMaker 클래스
    - BrideRandomNumberGenerator를 사용하여 건널 수 있는 다리 생성
  - BridgeGame 클래스
    - 게임 재시작/종료 기능
    - 다리 U/D중 이동 기능
    - 사용자가 이동한 다리와 생성된 다리 비교 기능
    - 종료 시 최종 게임 결과와 시도 횟수 출력
  - Bridge 클래스
    - 건널 수 있는 다리 가지고 있음
    - 외부에서 해당 값이 맞는지 물어보면 확인해주는 기능
  - MoveRecord 클래스
    - 이동한 기록 저장 클래스
    - up / down 각각의 Branch 클래스 가짐
    - 사용자가 이동하면 그 기록을 up/down Branch에 저장함
    - 문자열 변환 기능
  - Branch 클래스
    - 사용자 이동 기록을 저장하는 클래스
    - 리스트 요소로 실패 / 성공 중 하나를 가짐
    - 문자열 변환 기능
