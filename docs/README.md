## 구현 기능 목록
- --
- 게임 시작 및 종료 문구 출력


- 다리 길이 입력
    - 숫자 입력 확인
    - 3 이상 20 이하 값 확인


- 다리 생성
    - 길이(칸) 별 0과 1 무작위 값 설정


- 이동할 칸 입력
    - 문자(U와 D) 입력 확인


- 칸 이동
    - U 선택 시 위칸으로 이동
    - D 선택 시 아래칸으로 이동
    - 건널 수 있으면 O, 건널 수 없으면 X


- 게임 재시작/종료 입력
    - 문자(R과 Q) 입력 확인


- 잘못된 사용자 입력값을 경우, 에러 발생 및 다시 입력
