# 미션 - 다리 건너기

## 기능 목표
1. 사용자에게 다리 길이를 입력받는다.
2. 다리 길이의 타당성을 검증하고 길이에 맞게 다리를 랜덤으로 생성한다.
3. 이후 매 라운드마다 사용자에게 이동할 칸을 입력받는다.
   1. 이동할 칸이 정답이면 계속 진행한다.
   2. 이동할 칸이 정답이 아니면 게임을 다시 시도할지 여부를 입력받는다.
   3. 사용자가 재시도를 입력할 경우 생성된 다리는 바뀌지 않고 처음부터 다시 이동한다.
   4. 종료를 입력할 경우 결과를 출력하고 종료한다.
4. 모든 라운드에 정답을 입력하여 다리의 끝에 도달할 경우 게임이 종료된다.
5. 최종 게임 결과를 출력한다.
6. 이어서 게임의 성공 여부와 총 시도 횟수를 출력 후 프로그램이 종료된다. 

## 예외 사항
1. 다리의 길이가 3~20이 아닌 경우
2. 입력 형식이 주어진 주건과 다를 경우
    1. U : 위, D : 아래, R : 재시도, Q : 종료
