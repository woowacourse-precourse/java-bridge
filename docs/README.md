# 🎯 프로그램 흐름도

- 다리 길이 입력
- 길이에따른 랜덤 다리 생성
- 게임 시작 (실패시 재시도,종료 여부 판단)
- 현재까지 이동한 칸 출력
- 결과 출력


# 🚀 기능 구현 목록
### 입력 기능
- 다리 길이 입력 기능
- 게임 재시작 여부 입력 기능
- 이동할 칸 입력기능

### 출력기능
- 현재까지 이동한 칸에 따른 출력 기능
    - 올바른 칸으로 갔을 경우
    - 틀린 칸으로 갔을 경우
- 최종 게임 결과 출력 기능

### 컨트롤러 기능
- 랜덤 다리 생성 기능
- 게임 성공,실패 판단 기능

### 입력 예외 처리
- 다리 길이 입력시 3~20 사이 정수 아닐경우
- 게임 재시작,종료 여부 입력시 R,Q 가 아닌 문자 입력 했을 경우
- 이동할 칸 입력시 U,R 이 아닌 문자 입력 했을 경우
