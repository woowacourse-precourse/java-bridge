# 기능 구현 목록

## 1. 게임 시작

- 최초 시작 문구를 출력 한다.

## 2. 다리 길이 입력 받기

 - 다리 길이 입력 문구를 출력한다.
 - 다리 길이는 숫자만 입력 받을 수 있다.
 - 다리 길이는 3 ~ 20 사이의 숫자이다.
 - 다리 길이는 자연수만 가능 하다.

## 3. 다리 생성

 - 입력 받은 길이의 다리를 생성 한다.
 - 다리는 위아래 두 줄이 존재하며, 건널 수 있는 위치가 존재한다.

## 4. 다리 건너기

 - 이동할 칸 입력 문구를 출력 한다.
 - 이동할 칸을 입력 받는다.
 - 다리를 무사히 건널 때 까지 반복한다.

### 4.1 무사히 건너지 못한 경우
  
 - 다시 시도 여부 입력 문구를 출력 한다.
 - 다시 시도 여부를 입력 받는다.

### 4.2 다리를 건넌 경우

- 게임 성공 여부를 업데이트 한다.

## 5. 게임 종료

- 최종 게임 결과를 출력 한다.
- 게임 성공 여부를 출력 한다.
- 시도한 횟수를 출력 한다.
