## 기능 목차
//1.입력: 다리 길이
//반복 
//2.다리 생성: 무작위 0,1로 생성  
//3.사용자 이동: 다리길이반큼 반복  
//1)입력: 이동할 칸  
//2)만약 U 이고 윗 다리가 1 이면: O  
//3)만약 U 이고 윗 다리가 0 이면: X  
//반복 중단  
//4)만약 D이고 아랫 다리가 1이면: X
//5)만약 D이고 아랫 다리가 0이면: O  
//반복 중단  
//4.입력: 다리를 못건넜으면 다시 시도 여부 (R,Q)  
//1)R이면 재시작  
//2)Q이면 중단, 세임 결과 출력  
//5.출력: 게임 성공 여부, 총 시도한 횟수 

## 확인 사항
1.총 시도 횟수 입력 예외처리  
2.다리 길이 입력 예외처리  
3.이동할 칸 입력 예외처리  
4.재시작 여부 입력 예외처리
