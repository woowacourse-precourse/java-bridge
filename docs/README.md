# 미션 - 다리 건너기

## 👨‍💻 기능 구현 목록


- 다리 길이를 입력 받는 기능 
  - [ERROR] 숫자가 아닌 문자 입력
  - [ERROR] 범위에 맞지 않는 입력
- 다리 생성 기능 
  - 건널 수 있는 칸 선택(0,1 랜덤으로 결정)
  
- 플레이어가 이동할 칸을 입력 받음
  - 위 칸은 U, 아래 칸은 D
  - [ERROR] U 혹은 D 이외의 입력
- 게임 결과를 출력하는 기능
- 게임 재시작/종료 결정 기능
  - [ERROR] R,Q 이외의 입력
  - 재시작 시 기존 다리 사용
  - 종료 시 게임 횟수 출력
