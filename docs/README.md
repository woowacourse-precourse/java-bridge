# 4주차 다리 건너기 미션
***

## 기능 목록
### 입력
- [x] 다리 길이 입력받기
  - [x] [예외] 정수가 아닌 값을 입력했을 경우

- [x] 라운드마다 이동할 칸 입력받기

- [x] 게임 재시작/종료 여부 입력받기

- [x] 에러 문구 출력 기능


### 출력
- [x] 다리 출력하기
  - [x] 이동 실패 시 X 출력하기
- [x] 게임 시작 문구 출력
- [x] 최종 게임 결과 출력
  - [x] 현재까지 건넌 다리 출력
  - [x] 게임 성공 여부 출력
  - [x] 총 시도한 횟수 출력
- [x] 에러 메시지 출력


### 다리 건너기 게임
- [x] 주어진 길이에 맞게 다리를 생성하는 기능
  - [x] [예외] 3이상 20이하의 정수가 아닌 경우

- [x] 플레이어를 이동시키는 기능
  - [x] [예외] U나 D가 아닌 경우

- [x] 게임을 재시작/종료하는 기능
  - [x] [예외] R이나 Q가 아닌 경우

- [x] 다리의 끝에 도달했는지 확인하는 기능
