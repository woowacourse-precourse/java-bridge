# 기능 목록

## Model(Domain)
### Brige
### Player
    - 건널 칸 선택

## View
### 입력
    - 다리 길이 입력받기
    : 3~20 이하
    - player 건널 칸 입력받기
    : U, D만 입력받기
    - 재시작이나 종료입력받기
    : R, D만 입력받기

### 출력
    - 게임 시작 문구
    - 이동 가능한 칸 출력
    - 게임 종료 문구
    : 최종 게임 결과,
      게임 성공 여부,
      총 시도 횟수 
    - 건널 수 있는 칸 표시하기

## Controller
### BrigeController
    - 다리 건너기
    - 다리 생성하기(다리길이)
        :무작위 값이 0인 경우 아래 칸, 1인 경우 위 칸이 건널 수 있는 칸이 된다.
### GameController
    - 게임 재시작하기
    - 게임 종료하기
    - 총 시도 횟수 기록하기
    - 이동 칸이
