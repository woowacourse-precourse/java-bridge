## 📃 기능 목록

### 0 - 1. 사용자의 입력 
* 다리 길이 입력
  * 3 ~ 20 숫자
* 라운드마다 플레이어가 이동할 칸을 입력
  * U(위 칸)
  * D(아래 칸)
* 게임 재시작/종료 여부를 입력
  * R(재시작)
  * Q(종료)

### 0 - 2. 예외 처리
* 잘못된 값 입력 시
  * IllegalArgumentException 발생 후 
  * "[ERROR]"로 시작하는 에러 메시지 출력
  * 그 부분부터 입력 다시 받음

### 1. 출력
* 게임 시작 문구
```
다리 건너기 게임을 시작합니다.
```
* 게임 종료 문구
```
최종 게임 결과
[ O |   |   ]
[   | O | O ]

게임 성공 여부: 성공
총 시도한 횟수: 2
```
