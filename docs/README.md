# 로또 미션 구현 목록

## ● 주요 구현 기능
- 입력 메시지 출력하는 기능
- 입력값 받는 기능
- 출력 메시지 출력하는 기능
- 결과값 출력하는 기능
- 에러 메시지 출력하는 기능


- 다리 랜덤으로 생성하는 기능
- 사용자 이동시키는 기능
- 사용자 이동 기록하는 기능
- 재시작하는 기능

- [x] 다리 길이 입력 받기

- [x] 예외 발생 시 재입력 받기

## ● 예외 상황
- 다리 길이가 정수가 아닐 때
- 다리 길이가 올바른 범위가 아닐 때
- 이동할 칸 문자가 올바른 값이 아닐 때
- 재시작 여부가 올바른 값이 아닐 때
