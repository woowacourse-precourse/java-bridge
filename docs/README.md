#기능 구현 목록


---


- [X] 숫자를 입력 받아 다리를 생성하는 기능   
     - [X] 숫자인지 확인하는 기능  
     - [X] 숫자가 3이상 20이하인지 확인하는 기능   
     - [X] 입력받은 숫자 만큼 다리를 생성하는 기능  
          - [X] 무작위 0 혹은 1로 배열을 생성하는 기능


- [X] 라운드마다 이동할 칸을 입력받는 기능
    - [X] 입력된 문자가 U 혹은 D 인지 확인하는 기능
    - [X] 입력된 문자대로 다리가 출력되는 기능


- [X] 다리를 건너는 기능
    - [X] 이동한 칸이 이동 가능한 칸인지 비교하여 확인하는 기능
    - [X] 남은 칸만큼 반복하는 기능


- [X] 이동할 수 없는 칸을 선택한 경우 재시작/종료 여부를 입력받는 기능
    - [X] 입력된 문자가 R 혹은 Q 인지 확인하는 기능
    - [X] 입력된 문자가 R 인 경우 같은 다리로 재시작 하는 기능
    - [X] Q 인 경우 종료 문구를 출력하고 종료하는 기능


- [X] 게임이 종료되는 기능
    - [X] 종료 문구에서 성공 여부를 출력하는 기능
    - [X] 종료 문구에서 총 시도한 횟수를 출력하는 기능
