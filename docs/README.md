# 기능 목록

- [ ] 다리 생성

  - [ ] 다리 길이 입력 받기

  - [ ] 다리 만들기

- [ ] 게임 진행

  - [ ] 이동할 칸 입력 받기
  - [ ] 움직이기
  - [ ] 게임 진행 상황 출력
    - [ ] 실패 시, 재시도 여부 입력 받기
    - [ ] 재시도

- [ ] 게임 결과 출력



### 예외 처리

- [ ] 다리 길이
  - [ ] 다리 길이는 숫자여야 한다.
  - [ ] 다리 길이는 3부터 20 사이의 자연수여야 한다.
- [ ] 플레이어의 커맨드
  - [ ] 이동하기
    - [ ] 이동은 U, D로 가능하다.
  - [ ] 재시작
    - [ ] 재시작은 R, 종료는 Q로 가능하다.
