## 전반적인 개요

1. 다리 생성기 (**BridgeMaker**)
    - size가 주어지면 다리의 건너기의 정답을 U, D로 이루어진 List로 만든다.
    - BridgeRandomNumberGenerator를 사용

   #### 테스트 목록
       1. 주어진 size와 동일한 사이즈의 리스트를 리턴한다.
       2. 주어진 "U"와 "D"로만 이루어져 있는 List이다.
       3. BridgeGenerator에서 size만큼 0 또는 1을 받아 1은 U 로 0은 D로 변환하여 List를 만든다.

-------------------------------------------------------------

2. 다리 게임 (**BridgeGame**)
    - 다리와 입력을 비교하며 성공 여부 반환한다.
    - 다리 게임의 성공 여부를 반환한다.
    - 총 시도 횟수를 반환한다.
    - 다리와 현재까지의 성공여부를 담은 List를 반환한다.
    - retry() 호출 시 시도횟수가 1회 상승한다.
    - retry()을 호출하면 이동 궤적이 초기화 된다.

   #### 테스트 목록
       1. move(input)할 경우 bridgeTrack에 input이 추가된다.
       2. retry()이후 getBridgeTrack을 하면 size가 0이다.
       3. retry()의 호출 횟수에 따라 시도 횟수가 리턴된다.(호출이 없었다면 1, 1회 호출시 2)
       4. isRecentMoveSuccessful() 호출 시 최근 이동이 정답이었는지 반환된다. 
       5. 현재 게임의 게임 성공 여부를 반환다.

--------------------------------------------------------------------

3. 입력을 받기 (**InputView**)
    - 다리 사이즈 입력받아 예외 확인 후 int로 리턴
    - 움직임을 입력받아 예외 확인 후 String으로 리턴
    - 게임 재시작 여부를 입력받아 예외 확인 후 String으로 반환한다.
   #### 테스트 목록
       1. 다리 사이즈 입력 시 숫자 이외의 입력이 있는 경우 예외 발생
       2. 다리 사이즈 입력 시 범위에서 벗어난 경우 예외 발생
       3. 움직임 입력 시 여러 글자가 입력 될 경우 예외 발생
       4. 움직임 입력 시 D나 U 이외의 입력이 있는 경우 예외 발생
       5. 재시작 여부 입력 시 여러 글자가 입력 될 경우 예외 발생
       6. 재시작 여부 입력 시 R나 Q 이외의 입력 있는 경우 예외 발생
       7. 다리 사이즈 입력 시 올바르게 반환 되는 지 확인
       8. 움직임 입력시 올바르게 반환 되는 지 확인
       9. 재시작 여부 입력 시 올바르게 반환 되는 지 확인

--------------------------------------------------------

4. 출력
    1. 입력 받기 위한 출력 (**Message** enum)
    2. 게임 상황에 대한 출력 (**OutputView**)
        - 다리 2개를 받는다.(정답, 지금상황)
        - 둘은 비교하여 현재 상태를 출력한다.
   #### 테스트 목록
       1. 성공한 다리의 출력
       2. 실패한 다리의 출력
       3. 진행과정의 다리출력

-----------------------------------------------------------

5. 전체적인 흐름 관리 (**GamePlay**)

----------------------------------------------------------

## 다리 건너기

--------------------------------------------------

### bridge maker

--------------------------------------------------

#### **BridgeMaker** 클래스

**BridgeNumberGenerator**에서 발생된 숫자로 "U"와 "D"로 이루어진 다리 생성

- 필드

| 이름                                                | 내용                              |
|---------------------------------------------------|---------------------------------|
| **BridgeNumberGenerator** *bridgeNumberGenerator* | 다리 숫자 생성기로 어떤 방식으로 다리를 만들지 정한다. |

- 메서드

| 이름                                                   | 내용                          |
|------------------------------------------------------|-----------------------------|
| List\<String> **BridgeMaker**(BridgeNumberGenerator) | 다리 생성 번호 방식을 입력 받는다.        |
| List\<String> **makeBridge**(int)                    | 입력받은 숫자의 크기의 다리를 만든다.       |
| void **addBridge**(List\<String>, int)               | 다리에 숫자를 String으로 변황하여 추가한다. |

--------------------------------------------------------------------------

#### **BridgeState** enum

다리의 상태에 관한 상수를 저장한 열거형 클래스

- 필드

| 이름             | 내용                        |
|----------------|---------------------------|
| *UP*           | 위를 의미하는 숫자와 기호를 저장하고 있다.  |
| *DOWN*         | 아래를 의미하는 숫자와 기호를 저장하고 있다. |
| int *MIN_SiZE* | 다리의 최소 사이즈                |
| int *MAX_SIZE* | 다리의 최대 사이즈                |

- 메서드

| 이름                           | 내용                    |
|------------------------------|-----------------------|
| String **getBridgeMark**()   | 해당 객체가 의미하는 기호를 반환한다. |
| String **getBridgeNumber**() | 해당 객체가 의미하는 숫자를 반환한다. |

-----------------------------------------------------------------------------

### game play

--------------------------------------------------

#### **BridgeGame** 클래스

다리 건너기 게임의 실행과 상태 확인을 구현한 클래스

- 필드

| 이름                           | 내용           |
|------------------------------|--------------|
| List\<String> *answerBridge* | 정답인 다리       |
| List\<String> *BridgeTrack*  | 현재 시행의 다리 상태 |
| int *tryCount*               | 총 시도 횟수      |

- 메서드

| 이름                                   | 내용                                 |
|--------------------------------------|------------------------------------|
| **BridgeGame**(List\<String>)        | 정답인 다리를 입력받아서 시도 횟수를 1로 초기화 하면서 생성 |
| void **move**(String)                | 입력받은 움직임을 *BridgeTrack*에 추가        |
| void **retry**()                     | 시도횟수를 증가시키고 **reset**을 호출한다.       |
| void **reset**()                     | *BridgeTrack*을 초기화                 |
| List\<String> **getBridgeTrack**()   | 현재 다리 상태인 *BridgeTrack*을 반환한다.     |
| boolean **isRecentMoveSuccessful**() | 최근 움직임이 정답 다리와 일치하면 true를 반환한다.    |
| int **getTryCount**()                | 총 시도 횟수를 반환한다.                     |
| boolean **isBridgeGameSuccess**()    | 게임이 성공으로 끝난 상태라면 true를 반환한다.       |
| List\<String> **getAnswerBridge**()  | 정답 다리인 *answerBridge*을 반환한다.       |

| void **printRequireSize**()                  | 사이즈의 입력을 요구하는 메세지 출력                         |

| **GameState** **movable**()                  | 현재 움직여야하는 상태에서의 게임 진행, 움직임을 입력받고 상태를 반환한다.   |

| void **printWantToRetry**()                  | 게임의 재시도 여부를 요구하는 메세지를 출력                     |
| boolean **isWantToRetry**(String)            | 입력이 재시도를 의미하면 true를 반환한다.                    |
| void **printGameResult**()                   | 게임의 최종 결과를 출력한다.                             |
