# 미션 - 다리 건너기

## 👨‍💻 기능 구현 목록

### 다리 생성 :

- ✅ 다리 길이 입력
    - ✅ [ERROR] 숫자가 아닌 문자 입력
    - ✅ [ERROR] 범위에 맞지 않는 입력
- ✅ 랜덤 값에 의해 건널 수 있는 다리 위치 생성

### 플레이어 :

- 라운드 마다 플레이어가 이동할 칸(U, D)를 입력
    - [ERROR] U 혹은 D 이외의 입력
- 게임 재시작/종료 결정 기능
    - [ERROR] R,Q 이외의 입력
    - 재시작 시 기존 다리 사용
    - 종료 시 게임 횟수 출력

### 게임 :
- 게임 결과를 출력하는 기능
