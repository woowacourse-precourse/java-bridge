# 💡기능 목록

## 1. 다리의 길이 입력

### 예외 처리

- 3 이상 20 이하의 숫자를 입력

## 2. 다리 무작위 값으로 생성

- 무작위 생성(1과 0으로)
- 위 칸을 건널 수 있는 경우 U, 아래 칸을 건널 수 있는 경우 D값으로 설정


## 3. 이동할 칸 입력(U or D)

### 예외 처리

- U나 D를 입력하지 않을 때

## 4. 이동한 값 판별

- 무작위 값과 이동할 값이 일치할 경우, 입력한 위치(U or D)에 O 표시
- 아닐 경우 입력한 위치에 X로 표시

## 5. 게임 재시작/종료 여부를 입력(R or Q)

- 게임 재시작/종료 여부를 입력
- 게임 결과 표시

### 예외 처리

- R이나 Q를 입력하지 않을 때

## 6. 게임 종료

- 다리를 끝까지 건널 경우, 성공 여부와 시도 횟수 표시
