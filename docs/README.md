## 🚀 기능 목록 
- 다리 길이 입력
  - 숫자인지 예외처리
  - 3 이상 20 이하인지 예외처리
- 다리 생성하기
  - 입력받은 길이만큼 생성됐는지 테스트
  - 'D' 또는 'U'만 구성됐는지 테스트
- 다리 이동 입력
  - 영어 대문자인지 예외처리
  - 1글자인지 예외처리
  - 'D' 또는 'U'인지 예외처리
- 다리 이동하기
- 이동에 따른 결과 출력
- 재시도 또는 종료 여부 입력
  - 영어 대문자인지 예외처리
  - 1글자인지 예외처리
  - 'R' 또는 'Q'인지 예외처리
- 게임 재시도하기
- 게임 종료 결과 출력
- 게임 시작 로직 구현
