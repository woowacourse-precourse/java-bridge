## 기능 요구사항 목록
- 다리 길이를 입력 받는다.
- 다리 길이에 맞춰 정답 다리를 생성한다.
- 다리 길이에 맞춰 건널 다리 2개를 생성한다.
- 이동한 칸을 입력 받는다
    - 위 - U, 아래 - D
- 정답 다리와 비교 한다.
  - 틀리면 다리에 X 를 넣는다.
    - 재시작 여부를 묻는다
      - 재시도 - R, 종료 - Q
  - 맞으면 다리에 O 를 넣는다
- 다리를 다 건넜다면 게임을 종료한다.
- 건너온 다리 출력과 게임 성공여부, 총 시도한 횟수를 출력한다.

## 예외처리
- 다리 길이를 입력 받는다.
    - 숫자인지 확인
    - 3~20 사이의 숫자인지 확인
- 이동할 칸을 입력 받는다
    - U, D인지 확인
- 재시작 여부를 묻는다
    - R, Q인지 확인
