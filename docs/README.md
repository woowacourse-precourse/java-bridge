다리 만들기

--- 
### 기능 목록

- [x] 사용자에게 다리 크기를 입력받는다.
- [x] 입력받은 크기만큼 다리를 만든다.

- [x] 사용자에게 이동할 칸을 선택받는다 .
- [x] 입력받은 이동할 칸과 생성한 칸하나를 비교한다.
    - [x] 정답의 경우의 결과를 출력한다 . 
    - [x] 오답의 경우의 결과를 출력한다.

- [x] 게임 재시작을 물어본다.
- [x] 게임 결과를 출력한다.
### 예외처리 
- [x] 3 부터 20 사이의 숫자가 아닌 값을 입력 받는 경우 .
- [x] 예시 알파벳이 아니 값을 입력 받는 경우
