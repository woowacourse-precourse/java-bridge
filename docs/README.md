# 입력(Inputview)
## readBridgeSize()
 
- 시작 메세지, 다리 길이를 입력 받음.
- 리턴값 String

## readMoving()
 
- 이동 메세지, 이동할 칸 입력 받음. (U-위 칸, D-아래 칸)
- 리턴값 String

## readGameCommand()

- 재시작, 종료 여부 메세지, 재시작/종료 입력 받음 (R-재시작, Q-종료)
- 리턴값 String

# BridgeMaker
## makeBridge()

- 추가 메소드 없이 makeBridge()만 만든다.
- U를 1, D를 0으로 enum을 만든다.
- bridgeNumberGenerator.generate()가 1, 0으로 나오면 이걸 U랑 D로 리스트에 추가.
