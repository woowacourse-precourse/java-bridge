# 다리 건너기
## 기능목록
* 입력 기능
  * 다리 길이 입력 기능
  * 이동 위치 입력 기능
  * 게임 종료 여부 입력 기능 
* 출력 기능
  * 현재까지 이동한 다리 상태 출력 기능
  * 게임의 최종결과 출력 기능
* 생성 기능
  * 다리 생성 기능
* 게임 진행 기능
  * 칸 이동 기능
  * 게임 종료 여부 기능
* 검사 기능
  * 3 이상 20 이하의 숫자인지 검사
  * 입력된 값이 숫자가 아닐 시 검사
  * 입력된 값이 U/D 와 다른 값이면 검사
  * 입력된 값이 R/Q 와 다른 값인지 검사
* 예외 처리 기능
  * 다리 길이 입력시 3 이상 20 이하의 숫자가 아닐시 예외 발생
  * 다리 길이 입력시 입력된 값이 숫자가 아닐시 예외 발생
  * 이동할칸 입력시 U/D 와 다른 값이면 예외 발생
  * 게임 재시작/종료 여부 입력시 R/Q 와 다른 값이면 예외 발생
