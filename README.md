# 🌉 미션 - 다리 건너기

로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다. 로또 1장의 가격은 1000원이다.

    - [ ] [예외처리] 숫자가 아닐 경우

```text

```

- [ ] 수동,자동 로또 수들을 인스턴스로 저장한다
    - [ ] [예외처리] 로또 티켓 수가 0 이하의 수일 경우
    - [ ] [예외처리] 수동로또 수가 구매한 로또 수보다 많을 경우

- [ ] 수동으로 구매할 로또 번호를 입력받는다
    - [ ] [예외처리] 입력 형식이 아래 예시와 같지 않은 경우

```text
수동으로 구매할 번호를 입력해 주세요.
8, 21, 23, 41, 42, 43
3, 5, 11, 16, 32, 38
7, 11, 16, 35, 36, 44
```

- [ ] 구매한 로또내역을 출력한다.
    - [ ] 이전에 구매한 수동 로또번호를 포함한다

```text
수동으로 3장, 자동으로 11개를 구매했습니다.
```

- [ ] 당첨 번호를 입력받는다.

```text
   지난 주 당첨 번호를 입력해 주세요.
   1, 2, 3, 4, 5, 6
```

- [ ] 보너스 볼을 입력받는다.
    - [ ] [예외처리] 숫자가 아닐 경우

```text
   보너스 볼을 입력해 주세요.
   7
```
